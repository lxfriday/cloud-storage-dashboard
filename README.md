### 搜索

- 搜索关键词不区分大小写
- 可以依据文件名中的关键字搜索文件
- 也可以依据文件夹名，直接搜索文件夹下面的所有文件（最多 1000 个）

## 开发

- `npm run ins` 安装依赖
- `npm run dev` 启动 react-app 应用服务器
- `F5` 启动 扩展调试
- 点击右下角【云存储管理进入页面】

## 构建&发布

- `npm run build` 进行 extension(ts) 和 前端(react-app) 的编译处理
- `npm run publish` 编译并打包(vsce 自动打包)
